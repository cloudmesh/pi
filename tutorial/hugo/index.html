3. Tabs

macOS/Linux/Raspberry Windows

This program is typically already running on macOS and Ubuntu, so you do not have to invoke it separately. Hence a simple add will do.

22

(ENV3) you@yourlaptop $ ssh-add

On windows, you have to start the agent first so that ssh-add can be used while typing in the gitbash terminal:

22
23

(ENV3) you@yourlaptop $ eval `ssh-agent`
(ENV3) you@yourlaptop $ ssh-add

These two commands will start the ssh-agent and add your key to it so it is cached for future use within the same gitbash terminal.
